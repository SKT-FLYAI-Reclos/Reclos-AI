## 프로젝트 소개

😆 사진 한 장으로 **간편하게** 피팅 사진 만들어 판매하자.<br>
😁 중고 의류 구매하기 전, 피팅 모델에 내 옷을 **직접** 입혀보자.

<br>

Reclos는 중고 의류 거래 플랫폼으로, 판매/구매 과정에서의 불편함을 가상 피팅 AI 기술로 해결하고자 합니다.

옷 사진 한 장만으로 피팅 모델을 생성해 판매 과정을 간소화하고, 구매 전 자신의 옷과 미리 코디해 볼 수 있는 기능을 제공합니다.
